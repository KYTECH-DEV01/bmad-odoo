# Add Team Workflow

## Overview
Guided workflow to create a fully-wired, BMAD-compliant team through architectural discovery and automated wiring.

## Steps

1. **Route** — Classify intent and route to the correct factory workflow
   → [step-00-route.md](../step-00-route.md)

2. **Scope** — Define team composition, agents, roles, and overlap detection
   → [step-01-scope.md](step-01-scope.md)

3. **Connect** — Design integration wiring (contracts, config, routing)
   → [step-02-connect.md](step-02-connect.md)

4. **Review** — Summary of all decisions for approval
   → [step-03-review.md](step-03-review.md)

5. **Generate** — Create all files via BMB delegation + integration wiring
   → [step-04-generate.md](step-04-generate.md)

6. **Validate** — End-to-end validation confirming everything works
   → [step-05-validate.md](step-05-validate.md)

## Activation

Start by loading [step-00-route.md](../step-00-route.md) to classify the contributor's intent.
